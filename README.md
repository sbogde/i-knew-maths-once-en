# I Knew Maths Once (English Edition)

This repository is a digital preservation and English translation of my original, handwritten Pure Mathematics notebooks from the University of Bucharest, beginning with the Algebra curriculum of October 1994. 

As part of a broader "Maths from the Attic" initiative, this project translates decades-old theorems, equivalence relations, and Lie algebra problem classes into clean, modern LaTeX. 

Having transitioned from rigorous pure mathematics to a 25+ year career as a BI and Web Applications Developer, this repository serves as a bridge between abstract mathematical foundations and strict software engineering workflows. The project is typeset using Overleaf with direct GitHub synchronization to maintain a robust, version-controlled mathematical archive.

## Project Structure
* `courses/`: Theoretical lecture notes (*Cursuri*).
* `seminars/`: Applied problem classes (*Seminarii*).
